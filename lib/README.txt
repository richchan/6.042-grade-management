This project requires the following libraries to work:

- activation.jar (JavaBeans Activation Framework)
- commons-math-2.0.jar (Apache Commons Mathematics Library)
- various Google GData libraries:
  - gdata-client-1.0.jar
  - gdata-client-meta-1.0.jar
  - gdata-core-1.0.jar
  - gdata-docs-3.0.jar
  - gdata-docs-meta-3.0.jar
  - gdata-media-1.0.jar
  - gdata-spreadsheet-3.0.jar
  - gdata-spreadsheet-meta-3.0.jar
  - google-collect-1.0-rc1.jar
- mail.jar (JavaMail, required by GData)
- opencsv-2.1.jar (OpenCSV)
